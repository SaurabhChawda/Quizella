# Quizella

Quizella is a Quiz App, where user can test their Knowledge based on the selected category.It has multiple pages,Which will help user to track their Progress.

## Pages

- Home page
- Rules Page
- Question page
- Result Page
- Report Page
- Login Page
- Signup Page
